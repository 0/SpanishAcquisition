GUI¶

Tools for providing applications with a graphical user interface.

Class hierarchy¶

  • global_store — GUI global store
  • tool.box — GUI toolbox

The entire GUI framework is built on wxPython, so all widget classes ultimately inherit from a wxPython class, such as wx.Frame, wx.Panel, or wx.Dialog.

The global store provides a container for the spacq-related global state for an entire application. It includes separate namespaces for devices, resources, and variables, as well as a single slot for a pulse program.

The GUI toolbox provides several GUI primitives which appear frequently, such as a MessageDialog and functions for saving and loading Python-pickled and CSV files with a graphical dialog. It is recommended to use Dialog rather than wx.Dialog directly, in order to simplify disposal.
